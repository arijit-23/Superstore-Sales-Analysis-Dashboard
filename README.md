# 📊 Superstore Sales Analysis Dashboard

## 📝 Project Overview
This project features an interactive **Power BI Dashboard** created to analyze the performance of a retail "Superstore." I transformed a raw dataset of over **5,900 transactions** into actionable insights regarding profitability, sales trends, and customer behavior.

## 🛠️ Tech Stack & Skills
* **Data Visualization:** Power BI Desktop
* **Data Cleaning:** Power Query (ETL)
* **Data Modeling:** Star Schema
* **Calculations:** DAX (Data Analysis Expressions)
* **Data Source:** Excel/CSV

## 💡 Key Business Insights
Based on the analysis of **$1.56M in Total Sales** and **$175K in Total Profit**:
* **Profitability Analysis:** Identified high-margin categories (Technology) vs. low-margin categories (Furniture).
* **Shipping Efficiency:** Analyzed shipping delays by calculating the gap between Order Date and Ship Date.
* **Payment Trends:** Visualized customer preferences across **Online, Cards, and COD** payment modes.
* **Regional Performance:** Discovered that the **West and East regions** contribute the highest revenue share.

## 📂 Project Structure
* `SuperStore Sales DataSet.csv`: Raw data used for analysis.
* `SuperStore_Analysis.pbix`: The Power BI file containing the dashboard.
* `Screenshots/`: Visual previews of the final report.

## 🔧 How to Replicate
1. Clone this repository.
2. Open the `.pbix` file in Power BI Desktop.
3. If the data path is broken, go to **Transform Data > Data Source Settings** and point it to the included `.csv` file.
